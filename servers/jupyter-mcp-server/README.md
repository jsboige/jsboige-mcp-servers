# Jupyter MCP Server

MCP server for Jupyter notebook operations.

## Description

This server provides MCP (Model Context Protocol) tools for interacting with Jupyter notebooks:
- Notebook management (create, read, update, delete)
- Kernel operations
- Code execution
- Conda environment management

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Configuration

The server can be configured via command line arguments or environment variables.
